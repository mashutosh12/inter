# Backend Authentication System

## Tech Stack
- Backend: Node.js + Express
- Database: MongoDB
- Password hashing: Bcrypt
- Auth token: JWT

## Features
- User registration
- User login with validation
- Password hashing
- JWT authentication
  

## Setup Instructions
1. Clone the repo
2. Run `npm install`
3. Fill your Mongo credentials in `.env`
4. Run `npm start`
